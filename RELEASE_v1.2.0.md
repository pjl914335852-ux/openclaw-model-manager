# 🎉 发财猫模型管理器 v1.2.0 发布

**发布日期:** 2026-03-06

## 🚀 新功能

### 💻 系统监控

从 Telegram 直接查看服务器状态，无需 SSH 登录！

**功能：**
- ⏰ 系统运行时间
- 📊 CPU 使用率（总计、用户、系统、负载）
- 🧠 内存使用（总计、已用、可用、空闲）
- 💾 磁盘使用（总计、已用、可用、使用率）
- 🤖 机器人进程信息（PID、内存占用）

**使用方法：**
- 发送 `/system` 命令
- 或点击主菜单的 "💻 系统监控" 按钮
- 点击 "🔄 刷新" 更新数据

**示例：**
```
💻 系统监控

⏰ 运行时间: 49 days, 13:13

📊 CPU 使用率:
• 总计: 0.7%
• 用户: 0.5%
• 系统: 0.2%
• 负载: 0.37 / 0.16 / 0.11

🧠 内存使用:
• 总计: 3.8Gi
• 已用: 1.3Gi
• 可用: 2.5Gi

💾 磁盘使用:
• 总计: 75G
• 已用: 30G (42%)

🤖 机器人进程:
• PID: 123456
• 内存: 45.2 MB

💡 系统运行正常

[🔄 刷新] [◀ 返回]
```

## 🎯 安装/升级方法

### 新安装
```bash
git clone https://github.com/pjl914335852-ux/openclaw-model-manager.git
cd openclaw-model-manager
chmod +x install.sh
./install.sh
```

### 升级到 v1.2.0
```bash
cd /root/model-manager
git pull
systemctl restart model-manager
```

### 验证版本
发送 `/start` 到机器人，查看主菜单是否有 "💻 系统监控" 按钮。
